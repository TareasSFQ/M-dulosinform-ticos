¿Qué es CMD?

El Símbolo del Sistema (CMD) es un interprete de líneas de comando del sistema operativo Windows.

CMD (acrónimo de Command) es el intérprete de la línea de comandos de los sistemas operativos Windows. Es el análogo a los Shells de Unix, como por ejemplo, Bash, y le da al usuario la opción de realizar operaciones usando comandos.

Comandos Básicos

Aquí puedes encontrar algunos ejemplos de comandos básicos de CMD:

CD Sirve para regresar al directorio anterior
REN Sirve para renombrar los archivos o directorios
CLS Sirve para limpiar la pantalla.
DEL Borra el archivo o carpeta que indiques tras un espacio.
DIR Muestra todo el contenido de un directorio (carpeta).
TREE Muestra el árbol de carpetas de la carpeta que le indiques.
MD Crea una carpeta o directorio.
MOVE Te permite mover un archivo de un directorio a otro.
DATE Permite ver o cambiar la fecha del equipo.
FIND Busca una cadena de texto en uno o más archivos.
EXIT Cierra la ventana de la consola de Windows.
HELP Muestra todos los comandos que hay disponibles, poniendo en cada uno una breve descripción en inglés.
FORMAT Mucho cuidado con este último comando básico, porque sirve para formatear la unidad
TYPE Te permite crear un archivo desde la propia ventana de comandos.

Formas de ingresar a Cmd

Forma 1:Usando el atajo de teclado

  • Mantén presionada la tecla de Windows en tu teclado y presiona la tecla R para abrir el cuadro de diálogo Ejecutar.
    		•
  • Escribe "cmd" en el cuadro de texto y presiona Enter o haz clic en "Aceptar".
    		•
  • Se abrirá una ventana del símbolo del sistema.
    		•

    Forma 2:Usando el menú inicio

  • Haz clic en el botón de inicio (generalmente en la esquina inferior izquierda de la pantalla).
    		•
  • Escribe "cmd" en el cuadro de búsqueda y presiona Enter.
    		•
  • Se abrirá una ventana del símbolo del sistema
