¿Qué es diseño web?

Esta asignatura se centra en la creacion de sitios web, los cuales puedan ser accesibles desde cualquier navegador con acceso a internet, puede incluir audio, video, texto y más información multimedia. Pueden existir varios tipos de páginas como:
Página web estática: La cual muestra contenido que no puede ser modificado. no presenta interraccion con el usuario.

Ej.
Página web de un negocio ya que los datos de esta estaran de forma permanente.

Página web Dinámica: Son aquellas que permiten la creacion de apolecaiones dentro de la propia página web, ofreciendo mayor interracción con los usuarios que acceden a ella. Su creación es mas completa. Pertenece a la web 2.0 ya que la información varia en el tiempo real deacuerdo a las opciones tomadas por el usuario.

Introduccion a Diseño Web

-Conceptos básicos-

URL Uniform Resource Locators

Son dirrecciones que son dadas a recursos únicos en la web. Cada URL apunta a un recurso único ya sea una página web, una imagen o video.

Protocolo IP

Conjunto de normas que pieden ser utilizando para comunicar y enlazar información necesaria en la wen, que permite identidicar equipo con su propia dirreción o espacio que ocupa en la red.
IP:Es una dirreción de 32 bits en la actualidad debido a lal cantidad de dispositivos existentes se complica la identificación, por tal motivo se crea o desarrolla el IPv6

HTML (Hiper text markup languaje)

Hipertexto es un sistema de organización y presentación de datos. Se basa en la vinculación de fragmentos textuales o gráficos.
Link:Es un elemento de un documento elctrónico que permite accder y vinvular hacia otro coumento hacia otra parte del mismo.

Etiquetas básicas

  1. <.html>: Define el documento como un documento HTML.
  2. <.head>: Contiene metadatos sobre el documento, como el título y enlaces a scripts y hojas de estilo.
  3. <.title>: Especifica el título del documento, que aparece en la pestaña del navegador.
  4. <.link>: Vincula el documento HTML con recursos externos como hojas de estilo CSS.
  5. <.style>: Contiene estilos CSS para el documento.
  6. <.body>: Define el cuerpo del documento, que contiene todo el contenido visible para el usuario.
  7. <.header>: Representa la sección de encabezado de un documento o sección.
  8. <.nav>: Define un bloque de navegación para enlaces.
  9. <.main>: Especifica el contenido principal del documento.
  10. <.section>: Define secciones temáticas en un documento.
  11. <.article>: Representa una unidad autocontenida de contenido.
  12. <.aside>: Contenido adicional que está relacionado indirectamente con el contenido principal.
  13. <.footer>: Contiene información de pie de página para un documento o sección.
  14. <.h1> a <.h6>: Definen encabezados, siendo <.h1> el de mayor importancia y <.h6> el de menor.
  15. <.p>: Define un párrafo.
  16. <.a>: Define un enlace hipervinculado.
  17. <.img>: Inserta una imagen.
  18. <.ul>: Define una lista desordenada.
  19. <.ol>: Define una lista ordenada.
  20. <.li>: Define un ítem dentro de una lista.
  21. <.span>: Define una sección en línea, utilizada para aplicar estilos a una parte del texto.
